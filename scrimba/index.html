Width and height

All elements have a width and a height.

We can control the width and the height very easily, but be careful when setting heights on elements, it can get you in trouble!

try to avoid setting a HEIGHT property because css will shrink and grow depending on the content but if you set the height, you can get errors

margins - just 1 attribute = all 4 sides, 2 attributes = top & bottom, 3 attributes = top, right & left, 4 attributs = trbl clockwise
margins are outside the border

padding is the same as margins but is INSIDE the border -- it really adds more background of the element
